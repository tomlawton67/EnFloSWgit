Mano Calib

 

If LabVIEW is not running Launch it and open the EnFlo Hub window

 

Click on the "Calibration Program" icon to open the calibration program which should then initiallised itself to show the most recent calibration that has been carried out.

 

 

Click on the "Run" arrow to show the calibration setup

 

 

If the "Calib file" box on the "Calibration" panel was left blank you will be prompted to choose a new or existing calibration file. The Calibration file created in this example is "/Users/Paul_Hayden/Calib_Files/Mano_Calib.txt"

Please Note. The layout of this dialog box will be platform dependent.

 

 

Press the green "OK" button when you are ready to move on

 

 

Press the green "OK" button when you are ready to move on

 

 

The calibration setup has now been completed and the program is now ready to take the first calibration point. In this example the micromanometer we are calibrating is connected to a pitot-static tube in the wind tunnel and so we will be changing the pressure by changing the wind tunnel speed which at EnFlo can be done from the "Set Tunnel Speed Client" vi as shown below.

 

 

Type in the desired "Tunnel velocity m/s" and click on the "Run" or set the tunnel conditions manually

 

 

Click on the run arrow when the pressure transducer has settled at the first calibration value

 

 

Enter the micromanometer display value into the "Input Value Dialog.vi" panel, then press the green "OK" button

 

 

The calibration panel now shows the first calibration point. The voltage trace is shown in the plot labelled "sample of input signal" and the average voltage value of 0.064 Volts is recorded in the lower array directly above the entered value of 0.25 mmH2O.

The calibration constants A and B are NaN (Not a Number) since these cannot be evaluated from just one calibration point

 

 

Now the second tunnel speed can be entered then click on the "Run" arrow or set the next calibration pressure manually

 

 

Click on the "Run" arrow to take the next calibration point

 

 

Enter the front panel value into the input dialog box, then press the green "OK" button

 

 

The calibration panel now shows the first two calibration points.

The calibration constants A and B are now shown for the straight line fit for the equation Voltage = A + B * Pressure

The "Av % err" is the squareroot of the mean square error in pressure divided by the average pressure. Obviously for a two point calibration this value will be zero.
